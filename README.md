# Light Switch Project
Διαδραστική εφαρμογή που προσομοιώνει έναν διακόπτη φωτός με εφέ φακού και δυναμικές σκιές.

## Περιγραφή
Η εφαρμογή παρέχει:

- Flashlight Effect: Κίνηση φακού που ακολουθεί τον κέρσορα ή την αφή (Touch Events).

- Interactive Chain: Λειτουργικό σχοινί με drag-and-drop για εναλλαγή Dark/Light mode.

- Dynamic Shadows: Υπολογισμός κατεύθυνσης σκιάς στο κείμενο σε πραγματικό χρόνο μέσω JS.

- Mobile Responsiveness: Πλήρης προσαρμογή layout και λειτουργικότητας για κινητές συσκευές.

## Τεχνολογίες που χρησιμοποιήθηκαν
- JavaScript (Vanilla)
- CSS3
- HTML5

## Τοπική εκτέλεση
1. Ανοίξτε το αρχείο index.html στον browser.
2. Για τη σωστή εμφάνιση των assets, βεβαιωθείτε ότι υπάρχει ο φάκελος public με τα απαραίτητα αρχεία (PNG, JPG, MP3).
